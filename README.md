VantageBank DSA 🏦

Professional C++ Bank System with Binary Search Tree (BST) indexing.

Features

- BST Indexing: O(log n) search.
- Tiers: Automatic Bronze/Gold/Platinum status.
- Statements: Transaction history tracking.
- GUI: Built with wxWidgets.

Build

```bash
mkdir build && cd build
cmake ..
make
./bank_gui
```
