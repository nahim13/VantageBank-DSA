VantageBank DSA 🏦

Professional C++ Bank System with Binary Search Tree (BST) indexing.

Features

- AVL Tree Indexing: Self-balancing BST keeps account lookup at O(log n), even with sequential account numbers.
- Tiers: Automatic Bronze/Silver/Gold/Platinum status.
- Statements: Transaction history tracking.
- GUI: Built with wxWidgets.

Build

```bash
mkdir build && cd build
cmake ..
make
./bank_gui
```
